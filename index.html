Let's get started

Keep it stupid simple

  • With Fastest.io you can test all DOM elements of a webpage
  • If you call async functions, you don't have to type await as long as you're in the test function. Functions nested within don't have this, as it would break [].map
  • You can have more than one test in a single in a single javascript file and by doing it so you can have more than one test scenario

Use those browsers

  • With the directive --browser jsdom you can run tests in the jsdom environment, same for chrome
